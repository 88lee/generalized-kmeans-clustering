KMeansParallel

class KMeansParallel extends KMeansInitializer with SparkHelper

Initialize runs sets of cluster centers using the k-means|| algorithm. This is a variant of k-means++ that tries to find dissimilar cluster centers by starting with a random center and then doing passes where more centers are chosen with probability proportional to their squared distance to the current cluster set. It results in a provable approximation to an optimal clustering.

In this implementation, we allow the client to provide an initial set of cluster centers and closest distance for each point to those cluster centers. This allows us to use this code to find additional cluster centers at any time.
